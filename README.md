# Fluffy Mod Manager

Fluffy Mod Manager is a lightweight and easy-to-use RE Engine **game mod manager** that helps players install, organize, enable, disable, and manage mods for their favorite PC games. Whether you're a beginner exploring game modding or an experienced modder maintaining large mod collections, Fluffy Mod Manager provides a simple and efficient solution for mod management.

## Features

### Easy Mod Installation
Install and uninstall mods quickly with a streamlined interface designed for fast and reliable mod deployment.

### Mod Organization
Keep your mods organized with automatic sorting, categorization, and management tools.

### Conflict Detection
Identify mod conflicts before launching your game and avoid compatibility issues.

### Backup & Restore
Create backups of your installed mods and restore configurations whenever needed.

### Lightweight Performance
Fluffy Mod Manager is optimized for low resource usage while maintaining fast load times and smooth performance.

### Cross-Platform Support
Compatible with:

- Windows
- macOS
- Linux

## Installation Guide

### Windows Installation

1. Download the latest release from the Releases page.
2. Extract the downloaded `.zip` archive.
3. Open the extracted folder.
4. Launch `FluffyModManager.exe`.

### macOS Installation

1. Download the latest release.
2. Extract the archive.
3. Open Terminal.
4. Navigate to the extracted directory.
5. Run `FluffyModManager`.

### Linux Installation

1. Download the latest release.
2. Extract the archive.
3. Open a terminal window.
4. Navigate to the installation directory.
5. Run:

```bash
./FluffyModManager
```

## How to Use Fluffy Mod Manager

1. Launch Fluffy Mod Manager.
2. Open **Settings** and select your game directory.
3. Import or drag-and-drop mod files into the application.
4. Enable desired mods using the **Activate** button.
5. Manage load order if supported by your game.
6. Launch the game directly through Fluffy Mod Manager.
